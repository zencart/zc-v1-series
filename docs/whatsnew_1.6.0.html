Whats New ... Changes from v1.5.5 to v1.6.0

The following improvements and bugfixes are included in v1.6.0 since v1.5.5:


Admin side new features:
  • Admin LEAD pages for reports and utilities
  • Dashboard widgets on the Admin page
  • Improved appearance for many admin pages
  • Ajax support in the admin
  • Template system in the admin, which sets the stage for restyling and modernizing the admin console
  • Spiffycal replaced with a more modern jQuery plugin
  • Numerous improvements to Coupon capabilities
  • New Reports:
    • find duplicate model numbers
    • showing differences from a default installation
  • Admin configure.php file no longer used
  • Consolidation of functions from admin and catalog side
Catalog side new features:
  • Catalog LEAD pages for reports and utilities
  • Added a middle-tier to the template structure, to allow for "shared" customizations which might apply to multiple templates
  • Support for externally-hosted downloadable virtual products (storable on AWS S3,Dropbox, etc.)
  • Product-stock "availability" and product-"condition" indicators available
  • Language files simplified to relocate locale-specific content into a new locale.php file
  • Products with single valued attributes (e.g. PDF-only downloads) can now be added to the cart from the listing page
  • shipping supports insurance calculations for carriers that offer it in their modules
  • orders remember the customer's language, so order updates are sent in that language
  • guest checkout and other checkout flows (no password required until end of checkout; optional abbreviated checkout for virtual+free purchases)
Common features:
  • Simplified looping over query results
  • Updated all files for PHP7 compatibility
  • Consolidated Javascript files to remove duplication

Developer Documentation on the new 1.6.0 features is available on the Zen Cart Documentation Site.

Fixes:
  • CHANGE-343 - Updates for even greater consistency with the logs folder rename
  • CHANGE-345 - Fix typo in whos_online legend
  • CHANGE-346 - Fix outdated language in configuration menu help texts
  • CHANGE-347 - Fix TRY currency code in paypal modules
