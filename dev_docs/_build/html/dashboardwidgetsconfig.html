Widget Configuration SettingsÂ¶

Any dashboard widget is allowed to have multiple configuration settings that can be used to modify the data displayed by the widget.

An example would be a wiget that displays data that can be filtered by a date range.

The information relating to these configuration settings is stored in a number of database tables.
